Cat Wars

A new take on a classic game.

Project Overview

Cat Wars is a web-based game inspired by the classic Connect4. The objective of the game is to be the first player to connect four of their colored discs in a row, either horizontally, vertically, or diagonally.

The game features a user-friendly interface with intuitive controls. Players can take turns dropping their colored discs into the grid, aiming to strategically create a winning sequence while blocking their opponent.

The game keeps track of the current player, displays the winner when a winning sequence is formed, and allows for restarting the game for a new round of play.

Cat Wars is built using HTML, CSS, and JavaScript (specifically DOM manipulation), providing an interactive and enjoyable gaming experience.
